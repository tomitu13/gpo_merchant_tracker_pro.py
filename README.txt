📖 HOW TO USE

STEP 1: Get Server Age
  - Open Grand Piece Online
  - Look at BOTTOM RIGHT corner of screen
  - You'll see something like: 12:34
  
STEP 2: Enter in App
  - Type "12:34" in the "Age (MM:SS)" field
  - Click "▶ START"

STEP 3: Get Notifications
  - You'll get alerts 2 minutes BEFORE events
  - You'll get alerts WHEN events happen
  - Notifications appear with sound!

That's all! 🎉

💬 DISCORD WEBHOOK SETUP (Optional)

Want notifications IN YOUR DISCORD?

1. Go to Discord Server Settings
2. Click "Webhooks" 
3. Click "Create Webhook"
4. Copy the URL
5. Paste it in app Settings ⚙️
6. Click Save

Now all events post to Discord! 🎉

💾 FILES CREATED

The app creates 2 files:

gpo_config.json
  - Stores your settings
  - Stores your Discord webhook
  - Stores your theme preference

gpo_log.txt
  - Shows all events that happened
  - Useful for checking history

Both files are in the same folder as the app.

🎮 EVENT TIMES

MERCHANT
- First spawn: 10 min after server starts
- Duration: 10 minutes
- Next spawn: 40 minutes later
- Times: 10, 50, 90, 130, 170... minutes

STORE

- Examples: 1:00, 1:30, 2:00, 2:30, 3:00...
